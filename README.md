## About

Todo List is a straightforward application to create, complete, and delete tasks. It is a great starter project for learning CRUD operations, front-end state management, and simple persistence.

## Getting Started

These instructions will get a copy of the project up and running on your local machine for development and testing purposes.

### Installation

Clone the repo:

```bash
git clone https://github.com/tramohammadi/Todo_list.git
cd Todo_list
pyhton3 -m http.server
```

## Usage

- Add a new task using the input at the top
- Click on a task to toggle completion
- Use the Edit button to change a task's text
- Use the Delete button to remove a task
- Use the filter buttons to view All / Active / Completed tasks
